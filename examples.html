Basic Examples

Hello World

A simple example that prints "Hello World" to the chat.

print("Hello World!");

Built-in Functions

Built in functions can be used to place blocks or spawn mobs

placeWall("stone", "north", 20, 10); // Places a wall of stone at player's position

placeBlock(0, -1, 0, "oak_planks"); // Places block relative to the player's position

Let x = 0;
Let i = 50;

While (x < i) {
    spawn("zombie", 2, 0, 2); // Spawns 50 zombies relative to the player
    x = x + 1;
}

Advanced Examples

Fizz Buzz

FizzBuzz can be recreated using a for loop

// Initialize counter and limit
Let x = 0;
Let i = 50;
                        
// FizzBuzz loop
For (x upto i)
    If (x % 15 == 0)
        print("FizzBuzz");
    Elif (x % 3 == 0)
        print("Fizz");
    Elif (x % 5 == 0)
        print("Buzz");
    else
        print(x);

Arithmetic

MCSLang supports basic arithmetic operations

Let x = 4;

Let y = x + 2;
print(y)

Let z = y - 1;
print(y + " - 1 equals: "); // Showcase concatination
print(z);
                        
Let a = z * 3;
print(z + " * 3 equals: ");
print(a);
                        
Let b = a / 2;
print(a + " / 2 equals: ");
print(b);
